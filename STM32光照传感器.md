# 基于 STM32 的智能光照监测与 LED 调光系统
(Smart Lighting System based on STM32)

## 📖 项目简介
本项目是一个基于 STM32F103C8T6 微控制器的嵌入式闭环控制系统。系统通过 BH1750 传感器实时监测环境光照强度，并根据当前工作模式（自动/手动）通过 PWM 技术动态调节 LED 亮度，实现智能节能照明。此外，系统具备串口上位机实时监控和硬件异常保护功能。

## ✨ 核心功能
* **高精度光照采集**：采用 BH1750 传感器，支持宽范围 (0-65535 Lux) 数字光强采集。
* **平滑无级调光**：基于定时器的高频 PWM 输出，实现 LED 亮度的精准控制与平滑过渡（步进消抖）。
* **双模式无缝切换**：
  * **自动模式 (Auto)**：基于环境光强反向补偿（环境越暗，LED越亮）。
  * **手动模式 (Manual)**：支持脱离传感器干预的固定亮度输出。
* **实时状态监控**：通过 USART 串口实时上传环境光强、当前 PWM 占空比及工作模式。
* **高鲁棒性设计 (Failsafe)**：内置 I2C 总线超时防卡死机制与传感器断线安全保护（断线自动维持 50% 基础照明）。

## 🧰 硬件环境与引脚分配

| 硬件模块 | 型号/参数 | STM32 引脚 | 功能说明 |
| :--- | :--- | :--- | :--- |
| **主控 MCU** | STM32F103C8T6 | - | 核心控制单元 (72MHz 主频) |
| **光照传感器** | BH1750 | **PB6 (SCL), PB7 (SDA)** | 软件模拟 I2C 通信接口 |
| **LED 调光输出** | 普通/大功率 LED | **PA1** | TIM2_CH2 复用推挽输出 (PWM) |
| **状态指示灯** | 核心板载 LED | **PC13** | 系统心跳灯 (500ms 翻转) |
| **模式切换按键** | 独立按键 | **PB0** | 上拉输入，带 10ms 软件消抖 |
| **串口调试模块** | USB 转 TTL | **PA9 (TX), PA10 (RX)** | 波特率 9600, 8N1，连接 PC 上位机 |

## 📁 软件工程目录结构
本项目基于 **STM32 标准外设库 (SPL V3.5.0)** 开发，采用分层架构设计：

```text
Project: STM32
├── ASM                     # 启动文件
│   └── startup_stm32f10x_md.s
├── Library                 # ST 官方标准外设库源文件
│   ├── stm32f10x_gpio.c
│   ├── stm32f10x_tim.c
│   ├── stm32f10x_usart.c
│   └── ... 
├── System                  # 系统内核与时钟配置
│   ├── core_cm3.c
│   └── system_stm32f10x.c
├── User                    # 用户业务逻辑与 BSP 硬件驱动层
│   ├── main.c              # 主循环与业务控制调度
│   ├── delay.c/.h          # SysTick 精确延时驱动
│   ├── usart.c/.h          # 串口通信与 printf 重定向驱动
│   ├── timer.c/.h          # TIM2 PWM 调光底层驱动

│   ├── key.c/.h            # 按键采集与消抖驱动

└── STM32f10x_it            # 中断服务函数
    └── stm32f10x_it.c
